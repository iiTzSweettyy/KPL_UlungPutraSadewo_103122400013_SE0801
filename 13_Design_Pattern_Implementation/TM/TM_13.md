## 📝 Jawaban Tugas Mandiri

Event Delegation adalah sebuah pola (design pattern) atau teknik dalam JavaScript di mana kita menambahkan satu buah event listener pada elemen induk (parent), alih-alih menambahkan banyak event listener pada setiap elemen anak (child) secara individu.

Teknik ini sangat bergantung pada konsep fundamental JavaScript yang disebut Event Bubbling, yaitu proses di mana sebuah event (seperti klik) yang terjadi pada elemen terdalam akan "menggelembung" (bubble up) dan memicu event yang sama pada elemen-elemen induknya di DOM (Document Object Model).


Analoginya :
Tanpa Event Delegation: Setiap karyawan harus berdiri di depan pintu depan sepanjang hari untuk menunggu paket mereka masing-masing. (Sangat tidak efisien dan membuang waktu).

Dengan Event Delegation: Perusahaan menyewa 1 Resepsionis (elemen induk). Semua paket diserahkan ke resepsionis, dan resepsionis akan memeriksa nama pada paket tersebut (event.target) lalu menyerahkannya ke karyawan yang tepat.

Contoh implementasinya :

Kita hanya menempelkan satu event listener pada elemen pembungkusnya, yaitu <ul>.

```javascript
  const menuList = document.getElementById('menu-list');

  menuList.addEventListener('click', function(event) {
    // Kita mengecek apakah elemen yang benar-benar diklik adalah <li>
    if (event.target.tagName === 'LI') {
      console.log('Kamu mengklik: ' + event.target.innerText);
    }
  });

```
Cara kerjanya:
Saat mengklik <li>, klik tersebut akan merambat naik (bubble up) ke <ul>. Event listener pada <ul> akan menangkapnya, lalu kita menggunakan event.target (elemen spesifik yang paling awal diklik) untuk memastikan bahwa yang diklik benar-benar sebuah <li> dan bukan area kosong di dalam <ul>.
